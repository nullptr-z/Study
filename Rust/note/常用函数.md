## as_deref

Rust 的 Option 和 Result 枚举中提供的一个方法，允许您将实现了 Deref trait 的类型的值转换为一个指向其目标类型的引用。

```
let a = Option<T>
let b = a.as_deref() // b:Option<&T>
```
